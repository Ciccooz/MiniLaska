◆ printHorizontalSeparator()

static void printHorizontalSeparator ( int  cellSide,
char  left,
char  intersection,
char  right 
)
static

Stampa i separatori in base ai parametri passati.

Stampa per primo, e per una sola volta, il parametro 'left'. Poi per ogni cella viene stampato il carattere HORIZONTAL_SIDE e alla fine della cella il parametro 'intersection', questo processo è ripetuto per tutta la lunghezza della griglia. Una volta finita la griglia verrà stampato il parametro 'right'.

Parametri
cellSideDimensioni della singola cella
leftCarattere da stampare per primo
intersectionCarattere che viene stampato in mezzo
rightCarattere stampato alla fine della griglia

◆ UICoordinatesToTower()

Tower UICoordinatesToTower ( PlayableBoard  board,
int  UICoords[2] 
)

Restituisce la torre che corrisponde alle coordinate date.

Converte le coordinate che rappresentano una cella della griglia dell'UI, in una torre della griglia utilizzata dal programma.

Nota
La griglia utilizzata dal programma non utilizza alcune celle, quindi le coordinate delle colonne sono diverse. Per selezionare la cella corretta, è necessario inserire la riga, senza modifiche, e la colonna, divisa per 2.
Parametri
boardLa griglia contentente le torri
UICoordsLe coordinate da cui prendere la torre
