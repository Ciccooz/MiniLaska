◆ UICoordinatesToTower()

Tower UICoordinatesToTower ( PlayableBoard  board,
int  UICoords[2] 
)

Restituisce la torre che corrisponde alle coordinate date.

Converte le coordinate che rappresentano una cella della griglia dell'UI, in una torre della griglia utilizzata dal programma.

Nota
La griglia utilizzata dal programma non utilizza alcune celle, quindi le coordinate delle colonne sono diverse. Per selezionare la cella corretta, è necessario inserire la riga, senza modifiche, e la colonna, divisa per 2.
Parametri
boardLa griglia contentente le torri
UICoordsLe coordinate da cui prendere la torre
